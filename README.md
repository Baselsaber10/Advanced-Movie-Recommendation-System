# Advanced Movie Recommendation System

This project builds a movie recommendation system using the MovieLens 1M dataset. The goal is to analyze user rating behavior and build a scalable recommendation pipeline using Apache Spark MLlib.

## Project Objectives

- Analyze user-movie rating interactions.
- Prepare and clean the MovieLens dataset.
- Explore movie genres, rating distributions, and user activity.
- Build a collaborative filtering recommendation model using ALS.
- Evaluate the recommendation model using RMSE.
- Present insights through visualizations.

## Dataset

The project uses the MovieLens 1M dataset.

The dataset contains:

- 1,000,209 ratings
- 6,040 users
- 3,883 movies

Main files:

- ratings.dat
- movies.dat
- users.dat

## Project Pipeline

1. Data loading
2. Data quality inspection
3. Data preprocessing
4. Exploratory data analysis
5. Dataset integration
6. Spark-based recommendation modeling
7. Model evaluation
8. Visualization and reporting
